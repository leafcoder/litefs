# 博客系统示例

一个完整的可视化博客系统，展示了 Litefs 的 Web 应用开发能力。

## 功能特性

- 📝 文章发布和管理
- 👤 用户登录/登出
- 🎨 美观的 HTML 界面
- 📱 响应式设计
- 💾 会话管理
- 🖼️ 静态文件服务

## 项目结构

```
03-blog/
├── app.py              # 主应用文件
├── static/
│   └── css/
│       └── style.css   # 样式文件
└── README.md
```

## 运行方式

```bash
python app.py
```

访问 http://localhost:8080 查看博客。

## 默认账号

- 用户名: `admin`
- 密码: `admin123`

## 可用页面

- `/` - 首页，显示最新文章
- `/posts` - 文章列表
- `/posts/{id}` - 文章详情
- `/posts/new` - 写文章（需登录）
- `/login` - 登录页面
- `/logout` - 退出登录

## 功能演示

### 1. 浏览文章

无需登录即可浏览所有文章。

### 2. 用户登录

点击导航栏的"登录"链接，使用默认账号登录。

### 3. 发布文章

登录后，点击导航栏的"写文章"链接，填写标题和内容后发布。

### 4. 查看文章详情

点击文章标题或"阅读更多"链接查看完整内容。
