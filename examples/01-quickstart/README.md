# 快速入门

最简单的 Litefs 应用示例，帮助您快速上手。

## 示例文件

- `quickstart.py` - 快速入门示例
- `site/` - 示例站点文件
  - `index.py` - 首页
  - `hello.py` - Hello World 示例
  - `json.py` - JSON 响应示例
  - `info.py` - 请求信息展示

## 运行示例

```bash
python quickstart.py
```

访问 http://localhost:8080 查看示例。

## 学习路径

1. 从这个示例开始，了解 Litefs 的基本用法
2. 查看 [02-basic-handlers](../02-basic-handlers/) 学习各种处理器
3. 查看 [03-configuration](../03-configuration/) 学习配置管理
4. 查看 [04-middleware](../04-middleware/) 学习中间件使用
5. 查看 [09-fullstack](../09-fullstack/) 学习完整应用构建
