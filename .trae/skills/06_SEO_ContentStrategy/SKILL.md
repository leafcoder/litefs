# SEO Content Strategy

## Overview

帮助一人企业制定 SEO 内容策略，通过系统的关键词研究、内容规划和长尾词布局，提升网站搜索引擎排名和有机流量。

## When to Use This Skill

- 创建新的落地页或博客文章
- 制定内容营销策略
- 进行关键词研究和竞争对手分析
- 优化现有内容的 SEO 效果
- 规划内容日历和内容矩阵
- 创建符合 SEO 最佳实践的产品描述

## Core Concepts

### 1. 关键词研究框架

**关键词分类体系：**

```
顶级关键词 (Head Terms)
├── 搜索量大、竞争激烈
├── 如：CRM软件、小程序开发
└── 用于品牌建设和流量入口

长尾关键词 (Long-tail Keywords)
├── 搜索量小但转化率高
├── 如：中小企业 CRM 软件推荐、小程序开发报价
└── 用于精准获客和内容渗透
```

**关键词研究工具推荐：**

- Google Keyword Planner（免费）
- Ahrefs Keywords Explorer
- SEMrush
- Ubersuggest
- AnswerThePublic（问题挖掘）

### 2. 内容规划流程

**步骤一：业务目标对齐**
- 明确内容目标（品牌曝光/获客/转化）
- 确定目标用户画像
- 识别用户购买旅程阶段

**步骤二：关键词地图绘制**
- 核心业务词（产品/服务相关）
- 行业词（垂直领域专业词）
- 问题词（用户痛点和疑问）
- 比较词（对比评测类）

**步骤三：内容金字塔构建**

```
顶层：Pillar Content (支柱内容)
├── 深度长文 3000+ 字
├── 覆盖核心主题
└── 如：CRM 软件完整指南

中层：Cluster Content (集群内容)
├── 1500-2000 字
├── 支撑支柱内容的子主题
└── 如：CRM 功能解析、CRM 选型指南

底层：Blog Posts (博客内容)
├── 800-1500 字
├── 长尾关键词覆盖
└── 如：如何选择 CRM、CRM 使用技巧
```

### 3. SEO 内容创作标准

**标题优化 (Title Tag)：**
```html
<!-- 格式：主关键词 + 修饰词 + 品牌名 -->
<title>中小企业 CRM 软件推荐 | 5款高性价比选择 - EXAMPLE</title>

<!-- 长度：50-60 字符 -->
<!-- 包含主关键词靠前放置 -->
```

**Meta Description：**
```html
<meta name="description" content="寻找适合中小企业的 CRM 软件？本文精选5款高性价比 CRM，详细对比功能、价格和适用场景，助您做出明智选择。">
<!-- 长度：150-160 字符 -->
<!-- 包含行动号召和关键词 -->
```

**内容结构 (H1-H6)：**
```markdown
# H1：文章主标题（包含主关键词）

## H2：主要章节（对应关键词地图的子主题）

### H3：章节要点（支持 H2 的具体内容）

#### H4：细节说明（根据需要添加）
```

**内容质量标准：**
- E-E-A-T 原则：Experience（经验）、Expertise（专业）、Authoritativeness（权威）、Trustworthiness（信任）
- 原创度 > 90%
- 信息完整性：覆盖用户搜索意图的 100%
- 可读性：Flesch Reading Ease 60-70

### 4. 长尾关键词布局策略

**内容中布局位置：**

```markdown
1. URL 路径：/blog/seo-keyword-strategy
2. H1 标题：SEO Keyword Strategy for Beginners
3. 首个 100 字：包含主关键词和 1-2 个同义词
4. H2/H3 子标题：包含变体关键词
5. 图片 Alt 文本：描述性且包含关键词
6. 结论段落：自然融入关键词
7. 相关问题（FAQ）：覆盖 People Also Ask
```

**内部链接策略：**
```
文章A（支柱内容）
  ├── 链接到 文章B（集群内容）
  ├── 链接到 文章C（集群内容）
  └── 链接到 产品页（转化页）

反向链接来源：
  ├── 集群内容 链接回 支柱内容
  └── 博客文章 链接到 相关集群/支柱
```

### 5. 一人企业内容运营策略

**时间分配建议（每周 10 小时）：**
```
关键词研究：2 小时
  └── 每周 2-3 个新关键词发现

内容创作：5 小时
  └── 1 篇深度文章（2000+ 字）
  └── 2-3 篇短文（800-1000 字）

优化更新：2 小时
  └── 每周 1-2 篇旧文更新
  └── 内部链接优化

数据分析：1 小时
  └── Google Search Console 审查
  └── 排名和流量监控
```

**内容日历模板：**

| 周次 | 内容类型 | 关键词目标 | 状态 |
|------|----------|------------|------|
| W1 | 支柱内容 | CRM 软件完整指南 | 进行中 |
| W2 | 集群内容 | CRM 功能对比 | 待开始 |
| W3 | 博客文章 | 如何选择 CRM | 待开始 |
| W4 | 博客文章 | CRM 使用教程 | 待开始 |

## Interactive Design Process

- **Clarify Requirements**: 在制定内容策略前，明确业务目标、目标用户和竞争环境
- **Feedback Loop**: 使用 `mcp-feedback-enhanced` 工具（如有）进行方向确认和策略选择

## Best Practices

1. **用户意图优先**：关键词排名不是目的，满足用户搜索意图才是核心
2. **内容差异化**：在红海领域寻找独特角度（案例、数据、观点）
3. **持续迭代**：根据数据反馈不断优化内容
4. **技术配合**：好内容需要技术 SEO 支持（速度、结构化数据）
5. **长期视角**：SEO 效果通常需要 3-6 个月见效

## Common Pitfalls

- 过度优化关键词密度（keyword stuffing）
- 忽视用户意图匹配
- 内容更新不及时
- 缺少内部链接策略
- 忽视移动端体验
